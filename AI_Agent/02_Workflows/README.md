# Workflows 说明

本目录保存多智能体协同工作流程。

## 工作流文件

```text
00_总工作流_从想法到投稿.md
01_目标期刊论文拆解工作流.md
02_参考文献拆解工作流.md
03_审稿循环工作流.md
04_Word文档低Token工作流.md
05_论文进度识别与最小调用工作流.md
06_导师审稿意见处理工作流.md
07_跨学科领域适配与材料清单工作流.md
```

## 使用原则

主控智能体先读取总工作流、跨学科领域适配工作流和论文进度识别工作流，再根据任务阶段读取对应专项工作流。

启动目标期刊拆解或参考文献拆解前，必须使用用户提供的详细本地文件夹路径，不得默认期刊、默认专业方向或默认资料目录。

若用户提供导师意见、审稿意见、批注文件或返修意见，必须先读取 `06_导师审稿意见处理工作流.md`，生成采纳矩阵，再分配给对应子智能体局部处理。

工作流用于控制顺序、边界和停止条件，不能替代真实文献、实验或仿真证据。
