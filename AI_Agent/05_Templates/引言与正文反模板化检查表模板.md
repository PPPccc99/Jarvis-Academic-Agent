# 引言与正文反模板化检查表模板

## 检查对象

```text
稿件名称：

检查智能体：
检查日期：
目标期刊：
研究领域：

```

## 抽象层级检查

| 检查项 | 结论 | 问题位置 | 修改建议 |
|---|---|---|---|
| 引言是否过早罗列参数、特征参量、评价指标或实验条件 | 通过 / 需改 |  |  |
| 是否可用上位概念替代长串并列名词 | 通过 / 需改 |  |  |
| 具体技术细节是否已后移到理论、方法、评价或实验章节 | 通过 / 需改 |  |  |

## 自然推导检查

| 检查项 | 结论 | 问题位置 | 修改建议 |
|---|---|---|---|
| 是否按照“背景-进展-不足-问题-工作”逐步展开 | 通过 / 需改 |  |  |
| 是否体现作者自己的问题意识 | 通过 / 需改 |  |  |
| 是否说明已有方法解决了什么、本文场景下还缺什么 | 通过 / 需改 |  |  |
| 是否避免把技术点、指标和结论一次性堆入同一段 | 通过 / 需改 |  |  |

## 模板化风险检查

| 检查项 | 结论 | 问题位置 | 修改建议 |
|---|---|---|---|
