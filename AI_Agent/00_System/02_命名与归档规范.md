# 命名与归档规范

## 每次召唤贾维斯的工作文件夹

每次用户在新对话或当前对话中说出“召唤贾维斯”后，必须立即在项目主文件夹 `{{PROJECT_ROOT}}` 下创建一个本轮专用工作文件夹。

命名规则：

```text
年.月.日（序号）_贾维斯智能体工作任务
```

示例：

```text
{{PROJECT_ROOT}}\2026.5.21（1）_贾维斯智能体工作任务
{{PROJECT_ROOT}}\2026.5.21（2）_贾维斯智能体工作任务
```

序号从 1 开始；若当天已有同名序号，则自动递增。

本轮所有新生成或处理得到的文件，默认保存到该工作文件夹，不直接散落到项目主目录。长期通用配置文件仍保存在 `{{PROJECT_ROOT}}\AI_Agent`。

建议子目录：

```text
00_启动信息
01_任务记录
02_输入材料索引
03_目标期刊拆解
04_参考文献拆解
05_证据库
06_选题与创新点
07_论文任务书
08_正文草稿
09_仿真与实验计划
10_结果分析
11_图表
12_润色与投稿检查
13_审稿意见
99_临时文件
```

## 文件命名

建议使用：

```text
日期_阶段_主题_版本.扩展名
```

示例：

```text
2026-05-21_目标期刊拆解_{目标期刊名}_v1.md
2026-05-21_选题创新点_{研究主题}_v1.docx
2026-05-21_审稿意见_选题创新点_v1.md
```

## 任务记录

每个阶段完成后，主控智能体应在本轮工作文件夹的 `01_任务记录` 中记录：

- 输入材料
- 调用智能体
- 主要结论
- 采用意见
- 未采用意见及原因
- 阻塞问题
- 下一步任务
